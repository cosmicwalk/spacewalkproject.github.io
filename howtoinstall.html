Repositories

Spacewalk provides RPM repositories in COPR.

yum copr enable @spacewalkproject/spacewalk-2.10


Repositories could be still added as a standard YUM repository

Manual repository setup

Repositories could be set up manually by using .repo files from our build system.

Spacewalk Build System
Our RPM packages are GPG signed. Visit the download area of the project wiki for more information and access to the Spacewalk 2.10 public GPG key. For more information on using this yum repository and installing Spacewalk, please visit the Spacewalk installation procedures on the developer wiki.
