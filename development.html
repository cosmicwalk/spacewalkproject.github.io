Development

Source repository

Spacewalk's source repository uses the git source control system and is located at GitHub:  (you will need git installed, yum install git)

git clone git@github.com:spacewalkproject/spacewalk.git

For information on how to work with our source repository, please visit the wiki download page.

All development of Spacewalk happens in the community and out in the open. Communication between developers occurs on the development mailing list and in IRC, documentation is stored on our wiki, and our source code repository and all changes to it are publicly available.

Mailing Lists & IRC

Much of the discussion about how Spacewalk is developed takes place on our spacewalk-devel@redhat.com development mailing list as well as our IRC channel. Learn more about our mailing lists and IRC.

Wiki

The Spacewalk developer wiki is our main source of developer documentation, notes, and planning documents. If you'd like to be able to contribute to our wiki, you'll need a GitHub account. You can submit Pull request to spacewalkproject/spacewalk-wiki repository. More details can be found here.
